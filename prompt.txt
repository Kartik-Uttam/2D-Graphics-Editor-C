Prompt:
Explain the final 2D Graphics Editor C code.

Response:
The final program is a menu-driven 2D Graphics Editor written in C. It uses a two-dimensional character array to represent a drawing canvas. The user can draw geometric shapes such as lines, rectangles, circles, and triangles using coordinates entered through the keyboard. The program also supports storing shapes as objects, listing objects, deleting objects, and modifying existing objects.

Code:

#include <stdio.h>
#include <stdlib.h>

#define WIDTH 80
#define HEIGHT 24
#define EMPTY '_'
#define PIXEL '*'
#define MAX_OBJECTS 50

char picture[HEIGHT][WIDTH];

Explanation:
The canvas size is fixed as 80 columns and 24 rows. The `picture` array stores the complete drawing. Initially all positions contain `_`, representing empty space. Whenever a shape is drawn, some positions are changed to `*`.

Prompt:
Why is the Shape structure used in this project?

Response:
The Shape structure is used to store all information related to a graphical object.

Code:

struct Shape {
int id;
int type;
int x1, y1, x2, y2, x3, y3;
int radius;
};

struct Shape objects[MAX_OBJECTS];
int objectCount = 0;
int nextId = 1;

Explanation:
Every object drawn on the screen needs to store information about itself. The `id` field stores a unique identification number. The `type` field stores the shape type:

1 = Line
2 = Rectangle
3 = Circle
4 = Triangle

The coordinate fields store the position of the object. The radius field is used only for circles. Using structures allows multiple pieces of related information to be grouped together.

Prompt:
Explain how line, rectangle, circle and triangle are drawn.

Response:
Each shape has a separate function.

Code:

void drawRectangle(int x1, int y1, int x2, int y2) {
drawLine(x1, y1, x2, y1);
drawLine(x2, y1, x2, y2);
drawLine(x2, y2, x1, y2);
drawLine(x1, y2, x1, y1);
}

void drawTriangle(int x1, int y1, int x2, int y2, int x3, int y3) {
drawLine(x1, y1, x2, y2);
drawLine(x2, y2, x3, y3);
drawLine(x3, y3, x1, y1);
}

void drawCircle(int cx, int cy, int radius) {
int x, y;
int value;

```
for (y = -radius; y <= radius; y++) {
    for (x = -radius; x <= radius; x++) {
        value = x * x + y * y;

        if (value >= radius * radius - radius &&
            value <= radius * radius + radius) {
            setPixel(cx + x, cy + y);
        }
    }
}
```

}

Explanation:
`drawLine()` draws a line between two points. `drawRectangle()` uses four lines to form a rectangle. `drawTriangle()` connects three points using three lines. `drawCircle()` checks points around the center using the circle equation `x² + y² = r²` and plots points near the circumference.

Prompt:
Explain redrawPicture().

Response:
`redrawPicture()` clears the canvas and draws all stored objects again.

Code:

void redrawPicture() {
int i;

```
clearPicture();

for (i = 0; i < objectCount; i++) {
    if (objects[i].type == 1) {
        drawLine(objects[i].x1, objects[i].y1,
                 objects[i].x2, objects[i].y2);
    }
    else if (objects[i].type == 2) {
        drawRectangle(objects[i].x1, objects[i].y1,
                      objects[i].x2, objects[i].y2);
    }
    else if (objects[i].type == 3) {
        drawCircle(objects[i].x1, objects[i].y1,
                   objects[i].radius);
    }
    else if (objects[i].type == 4) {
        drawTriangle(objects[i].x1, objects[i].y1,
                     objects[i].x2, objects[i].y2,
                     objects[i].x3, objects[i].y3);
    }
}
```

}

Explanation:
Instead of manually erasing one shape, the program clears the whole canvas and redraws every stored object. This is useful after delete and modify operations.
